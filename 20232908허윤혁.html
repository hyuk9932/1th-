♥ HTML 태그 살펴보기 ♥

  • <title>
  • <h1> ~ <h6>
  • <hr>
  • <p>
  • <div>
  • <span>
  1. <title>

    2. 브라우저의 타이틀바에 나타낼 내용을 입력하는 태그이다.head 부분에 기술해야 한다

  2. <h1>

    3. 제목 글자에 사용하는 태그이다.글자를 진하게 나타내고 제목 위,아래에 빈 줄을 삽입해준다.

  3. <hr>

    4. 회색의 음영이 있는 수평선을 좌측 끝부터 우측 끝까지 그어 준다.단락 구분을 위해 주로 사용한다.

  4. <p>

    5. 문단을 만들 때 사용하는 태그이다.문단의 앞,뒤에 줄을 바꾸고,빈 줄을 삽입해 준다.
    <p>태그를 기준으로 앞,뒤 문단 구분만을 목적으로 사용할 땐 종료 태그를 쓰지 않아도 된다.
    스타일 시트로 문단을 꾸미는 경우 반드시 종료 태그를 사용해야 한다.

  5. <div>

    6. 여러 태그들을 블록으로 묶을 때 사용하는 태그이다.
    블록 전체에 동일한 CSS 스타일을 적용하거나 자바스크립트로 블록을 하나의 단위처럼 다루고자 할 때 사용한다.

  6. <span>

    7. 텍스트 일부분을 묶을 때 사용하는 태그이다.
    텍스트 일부분에 별도의 스타일 시트를 적용하거나 자바스크립트 코드로 텍스트 일부분을 제어하고자 할 때 사용한다
